#DigiPet

DigiPet is a web-based pet care reminder system that helps pet owners stay on top of their daily pet care responsibilities. Users can create pet profiles, schedule reminders, and receive automated email notifications so no important task is ever missed.

Features 

- User registration and login with password strength validation
- Individual pet profiles with profile pictures
- Email-based reminder scheduling with automated notifications
- Reminder archiving and recovery
- Past reminders visually highlighted in grey
- Blog post creation and management
- Admin dashboard for managing users and blog content
- Profile picture upload via Cloudinary
- Contact form for user support

Technologies Used

- Backend — Python, Flask
- Frontend — HTML, CSS, JavaScript, Bootstrap
- Database — PostgreSQL hosted on Neon
- ORM — SQLAlchemy
- Email — Flask-Mail with Gmail SMTP
- Scheduling — APScheduler
- Image Hosting — Cloudinary
- Authentication — Flask-Login
- Deployment — Render
